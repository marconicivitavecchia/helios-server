## Marconi Civitavecchia
Questo progetto è un fork per le votazioni dell'Istituto Tecnico Marconi di Civitavecchia durante il periodo della pandemia COVID-19.

Il progetto è stato tradotto completamente in italiano e speriamo possa essere d'aiuto anche a tutte le scuole, università ed enti di lingua italiana.

Per informazioni non esitate a contattarci!
